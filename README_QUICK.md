# FastPayZone — Quick Setup Guide (Hindi)

यह फाइल आपकी मदद करती है कि Backend + Frontend + Domain कैसे Live करना है।  
सब कुछ बहुत आसान भाषा में लिखा है।  

---

# ✅ STEP 1 — ZIP को GitHub पर Upload करो

1. GitHub → New Repository बनाओ (नाम: fastpayzone)
2. "Add file → Upload files"
3. ये फाइलें अपलोड करो:
   - package.json  
   - server.js  
   - .env.example  
   - public/index.html  
   - public/admin.html  
   - data/ (empty folder नहीं चलेगा — इसे manually नहीं डालना)  

4. “Commit changes” दबा दो।

---
